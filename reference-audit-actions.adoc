= Actions de sauvegarde et de restauration auditées
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
La page d'audit de la NetApp Console enregistre les actions suivantes effectuées par les utilisateurs dans NetApp Backup and Recovery dans le cadre de sa fonction d'audit. Pour plus d'informations sur la consultation et le filtrage des événements d'audit, reportez-vous à  https://docs.netapp.com/us-en/console-setup-admin/task-monitor-cm-operations.html#audit-user-activity-from-the-audit-page["Audit de l'activité des utilisateurs depuis la page d'audit"^]la documentation de la Console.

Pour chaque action auditée, la Console enregistre le nom du service, l'agent, la ressource, l'utilisateur et l'état de l'action.
